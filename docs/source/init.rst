Краткое описание
----------------

Данный модуль создаёт нужные для сервера приложений объекты класса Flask, SQLAlchemy, 
конфигурирует некоторые другие вещи, создаёт все недостающие таблицы в БД, а также
запускает сам сервер приложений.
