Краткое описание
----------------

Данный модуль содержит модель базы данных, а точнее все таблицы и их связи между собой.
Модель базы данных описана в формате классов Python, благодаря библиотеке SQLAlchemy 
и каждая таблица наследуется от класса ``SQLAlchemy.Model``. Эта библиотека упрощает
процесс написания запросов в БД и является ORM-системой, которая после запроса помещает
собранные данные в объекты классов Python, с которыми внутри кода проще работать.

Также, таблица ``User`` использует также и другой класс, с которого наследуется, и этот класс
- ``UserMixin``. Этот класс содержится в библиотеке ``flask-login`` и как следует из названия нужен
для реализации класса пользователя для того, чтобы потом этот класс использовать для
авторизации и входа на сайт.

В этом модуле также используются некоторые переменные и объекты, созданные в ``__init__.py``.
